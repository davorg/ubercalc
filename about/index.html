Uber Fare Calculator

The Uber fare calculator is a simple piece of Javascript silliness that was built by Dave Cross. He is @davorg on Twitter and would love to hear if you find this page at all useful or if you have any suggestions for improvements.

The code is on Github and pull requests are very welcome.

Uber's London prices are taken from their web site.

The site is built using jQuery and Bootstrap and is hosted on Github Pages.
